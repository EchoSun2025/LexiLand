# 🚀 LexiLand Read - 快速启动指南

## 方式一：一键启动（推荐）⭐

### 首次设置：

1. **双击运行** `CreateShortcut.vbs`
   - 会在桌面创建一个带Logo的快捷方式
   - 名称：`LexiLand Read.lnk`

2. **以后每次使用**：
   - 双击桌面上的 **LexiLand Read** 图标
   - 自动启动服务器并打开浏览器
   - 就这么简单！✨

---

## 方式二：直接运行批处理文件

如果不想创建桌面快捷方式，也可以：

- 双击 `LexiLand.bat`
- 效果一样，会自动启动并打开浏览器

---

## 📝 使用说明

### 启动后会发生什么：

1. ✅ 启动后端服务器（3000端口）
2. ✅ 启动前端服务器（5173端口）
3. ✅ 6秒后自动在浏览器打开应用
4. ✅ 可以开始使用了！

### 如何停止：

- 关闭弹出的两个 PowerShell 窗口
- 或运行 `stop-dev.ps1`

---

## 🌐 访问地址

- **前端应用**: http://localhost:5173
- **后端API**: http://localhost:3000
- **健康检查**: http://localhost:3000/health

---

## ⚠️ 常见问题

### 问题1：快捷方式没有显示Logo

**原因**：Windows批处理文件(.bat)不支持PNG图标

**解决方案**：
1. 将Logo转换为 `.ico` 格式
2. 或者接受使用默认的批处理文件图标

### 问题2：浏览器没有自动打开

**可能原因**：
- 服务器启动时间超过6秒
- 端口被占用

**解决方案**：
- 手动打开浏览器，访问 http://localhost:5173
- 检查端口是否被占用

### 问题3：启动失败

**检查**：
- 确保已运行过 `npm install`
- 确保 `.env` 文件配置正确
- 查看PowerShell窗口的错误信息

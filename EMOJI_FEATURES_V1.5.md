# Emoji 功能完整升级 - v1.5

## 更新日期
2026-03-11

## 三大功能升级

### 1. ✅ 显示生成模型信息

**功能**：在单词卡上显示用于生成图片的 AI 模型名称

**实现**：
- 后端返回 `model` 字段（如 `gpt-image-1-mini`, `gpt-image-1`, `dall-e-2`）
- 数据库添加 `emojiModel` 字段存储模型信息
- WordCard 显示"Generated by: {model}"标签

**效果**：
```
Generated by: gpt-image-1-mini  (成本: $0.005)
Generated by: gpt-image-1      (成本: $0.011)
Generated by: dall-e-2         (成本: $0.020)
```

### 2. ✅ 修复 Emoji 隔离问题

**问题**：之前所有单词共享同一个显示状态，导致生成新图片时覆盖所有单词的 emoji

**解决方案**：
- 每个单词的 emoji 完全独立存储在数据库中
- 新增字段：
  - `emojiImagePath`: AI 生成的图片路径
  - `emojiModel`: 使用的生成模型
  - `manualEmoji`: 用户手动选择的 emoji

**优先级逻辑**：
```
手动选择 emoji (manualEmoji)
    ↓ (如果没有)
AI 生成图片 (emojiImagePath)
    ↓ (如果没有)
自动推荐 emoji (emojilib 算法)
```

### 3. ✅ 右键选择 Emoji

**功能**：右键点击 emoji 图标，打开选择器手动选择喜欢的 emoji

**操作说明**：
- **左键点击**：生成 AI 图片（原有功能）
- **右键点击**：打开 emoji 选择器

**选择器特性**：
- 250+ 常用 emoji（表情、动物、物体、天气等）
- 10x25 网格布局
- 悬停效果
- 点击即选，自动保存到数据库

**数据存储**：
- 手动选择的 emoji 保存到 `manualEmoji` 字段
- 优先级最高，会覆盖 AI 生成的图片

## 使用场景

### 场景 1：使用 AI 生成

1. 左键点击 emoji
2. AI 生成个性化图片
3. 自动显示模型信息（如 `gpt-image-1-mini`）
4. 图片和模型信息保存到数据库

### 场景 2：手动选择

1. 右键点击 emoji
2. 从选择器中选择喜欢的 emoji
3. 显示"✓ Manually selected emoji"
4. emoji 保存到数据库（优先级最高）

### 场景 3：切换

1. 已有 AI 图片，想换成手动 emoji → 右键选择
2. 已有手动 emoji，想生成 AI 图片 → 左键生成（会覆盖）

## 技术实现

### 状态管理

```typescript
const [displayedEmoji, setDisplayedEmoji] = useState<string>('');
const [generatedModel, setGeneratedModel] = useState<string | null>(null);
const [showEmojiPicker, setShowEmojiPicker] = useState(false);
```

### 优先级判断

```typescript
useEffect(() => {
  if (annotation.manualEmoji) {
    setDisplayedEmoji(annotation.manualEmoji);
  } else if (annotation.emojiImagePath) {
    setDisplayedEmoji(annotation.emojiImagePath);
  } else {
    setDisplayedEmoji(getWordEmoji(annotation));
  }
  
  if (annotation.emojiModel) {
    setGeneratedModel(annotation.emojiModel);
  }
}, [annotation.word, annotation.emojiImagePath, annotation.manualEmoji, annotation.emojiModel]);
```

### 事件处理

```typescript
// 左键：AI 生成
const handleEmojiClick = async () => {
  const result = await generateEmojiImage(...);
  await updateEmojiImagePath(word, imagePath, model);
};

// 右键：打开选择器
const handleEmojiRightClick = (e: React.MouseEvent) => {
  e.preventDefault();
  setShowEmojiPicker(!showEmojiPicker);

// 选择 emoji

  await updateManualEmoji(word, emoji);

```
